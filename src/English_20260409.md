**English Corner:**

你的訊息：`"Why the response is {...}, but the web page doesn't show those data? only show '今日尚無資料（非交易時段或假日）'"`

整體語意清楚，有一個小修正：

| 原句 | 修正 |
|---|---|
| `doesn't show those data` | `doesn't show the data` |

**為什麼？** `data` 在英文中通常作為不可數名詞使用，不加 `those`。更自然的說法是 `the data` 或 `any of the data`。

完整修正版：
> **"Why does the response contain `{...}`, but the web page doesn't show the data? It only shows '今日尚無資料'."**

整體英文表達很清楚，問題描述得很精準！
