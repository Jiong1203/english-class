> Session: `57a5c182` · 08:48:41

**English Corner:**

您寫的：
> "If user use wrong password to login, the system doesn't show the error message in frontend."

| 您的寫法 | 建議修正 |
|---|---|
| If user **use** wrong password | If **a** user **uses** the wrong password |
| to login | to **log in** |

**說明：**
- `user` 是單數可數名詞，前面需要冠詞 `a`，動詞也要用第三人稱單數 `uses`。
- `login` 當動詞使用時，美式英文中寫成 `log in`（兩個字）；`login` 是名詞或形容詞（如 `login page`）。

**修正後：** "If a user uses the wrong password to log in, the system doesn't show the error message on the frontend."
